HTML Cheat Sheet

Tag Description
< p > Semantic element representing a paragraph
<header> Semantic element representing a pages header
<nav> Semantic element representing a pages navigation section
<figure> Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
<section> Defines a section in a document. A section is a thematic grouping of content, typically with a heading.
<article> Specifies independent, self-contained content. An article should make sense on its own.
<aside> Semantic element representing a pages header
<footer> Semantic element representing a pages footer

Table Attributes

Tag Description
<table> The wrapper element for all HTML tables.
<thead> The set of rows defining the column headers in a table.
<tbody> The set of rows containing actual table data.
<tr> The table row container.
<td> The table row container.
<tfoot> The set of rows defining the footer in a table.
